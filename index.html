Sobre

“Wissen” trata-se de uma Single Page Application (SPA). Um tipo específico de aplicação web, planejada para que a sua experiência de usuário seja similar a de um aplicativo “Desktop”. A ideia é construí-la usando a linguagem de programação Javascript, em conjunto com a biblioteca React. A escolha foi feita analisando o mercado, React é amplamente utilizada e é ótima em desempenho. O editor de texto será construído com base na API do Google docs, criando uma experiência familiar, porém melhorada.

A arquitetura de desenvolvimento será o modelo MVC — Model, View, Controller — muito comum no mercado devido a sua alta manutenibilidade. Todos os livros não públicos ficarão encriptados usando o modelo DES — Data Encrypt Standard. O autor do livro poderá solicitar acesso a chave de encriptação caso queira compartilhar seu trabalho de maneira gratuita para algum avaliador.
